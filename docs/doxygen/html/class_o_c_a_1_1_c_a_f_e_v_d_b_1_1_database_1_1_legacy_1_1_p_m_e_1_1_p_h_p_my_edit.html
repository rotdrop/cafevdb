◆ makeTimeStampFromDatabase()

makeTimeStampFromDatabase ( $databaseValue)
protected

{}

Create a Unix time-stamp from the data-base.

Strategy: if the date/time value in the data base does contain a time, then assume UTC and just convert to a time-stamp.

If the value in the data-base does not contain a time then adjust to 00:00 local-time by shifting with the time-zone offset of the local user.
