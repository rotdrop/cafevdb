Detailed Description

An entity listener for Entities\ProjectEvent.

The task is to remove the absence category from the associated calendar event when removing the absence field entity from the event entity. The other way round (adding the absence field category when the absence field is set) need not be handled as the absence field is set as a reaction to setting the "record absence" category. However, the absence field itself may be removed through the UI in which case the absenceField property of the project-event is set to null. In this case we want also remove the absence category from the associated calendar event.
