◆ configured()

configured ( )

Return an array with necessary configuration items, being either true or false, depending the checks performed.

The summary component is just the logic and of all other items.

Returns
array
[ 'summary',
'usergroup',
'encryptionkey'
'orchestra',
'shareowner',
'sharedfolder',
'database',
]

where summary is a bool and everything else is array('status','message') where 'message' should be empty if status is true.
