Detailed Description

Class ScanFiles is a background job used to run the file scanner over the user accounts to ensure integrity of the file cache.

This is a copy of

OCA\Files\BackgroundJob\ScanFiles

which is triggered by a JS timer via Ajax from the front-end in order to get authenticated background jobs.
