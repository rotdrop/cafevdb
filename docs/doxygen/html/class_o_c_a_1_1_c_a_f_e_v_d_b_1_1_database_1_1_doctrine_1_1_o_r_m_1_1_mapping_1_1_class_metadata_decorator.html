◆ updateInverseSide()

updateInverseSide ( object $entity,
array $association,
object $targetEntity,
ClassMetadataInterface $targetMeta )
private

Update the inverse side of an association after auto-installing $targetEntity into $entity (the owning side).

This is only done if it does not produce reloads from the data-base, so only "real" entities, initialized proxies and initialized lazy collections are updated.

Parameters
object$entityThe current $entity which must be the owning side of the association.
array$associationThe association mapping of the current field (the field itself is not needed here).
object$targetEntityThe target-entity which must be the inverse side of the association.
ClassMetadata | ClassMetadataDecorator$targetMetaThe class meta-data of the target-entity.
Returns
void
