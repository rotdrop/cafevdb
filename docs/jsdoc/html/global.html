Brief UI check for invalid input elements.

Name Type Description
container jQuery

Either a container element containing a form or a form. The function checks all contained forms if cointainer is not itself a form element.

If the container does not contain a form and is not itself a form, then all contained elements are search for ':invalid'.
options function

Options with components 'cleanup', 'labelCallback', 'afterDialog' and 'timeout'.
