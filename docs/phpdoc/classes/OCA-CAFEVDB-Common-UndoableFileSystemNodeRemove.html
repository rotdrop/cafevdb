public __construct(string|callable $name[, bool $gracefully = false ][, bool $recursively = false ][, string $ignoredFiles = '/^[0-9]*-?README(.*)$/i' ][, string|null $nodeType = null ]) : mixed
Parameters
$name : string|callable
$gracefully : bool = false

Do not complain if folders are non-empty or do not exist.

$recursively : bool = false

Recursively remove folders. If not set only non-empty folders will be removed. Ignored if the given node points to a file.

$ignoredFiles : string = '/^[0-9]*-?README(.*)$/i'

A regular expression masking out ignored files by their name in the decision whether a directory is empty or not. It defaults to ignoring all variants of README. Ignored if the given path points to a file.

$nodeType : string|null = null

Only remove matching node type.
