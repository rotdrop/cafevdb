protected static array<string|int, mixed> $sql = [self::STRUCTURAL => [], self::TRANSACTIONAL => []]

Split the SQL commands into "structural" commands which cannot be wrapped into a transactions due to some "auto commit" "feature" of MySQL/MariaDB and/or the used PHP SQL-engine and "transactional" SQL commands which can be undone by wrapping them in to a transaction.

A removed entity will be removed from the database at or before transaction commit or as a result of the flush operation.

Parameters
