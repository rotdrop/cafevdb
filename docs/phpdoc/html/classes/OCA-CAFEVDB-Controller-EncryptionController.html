A removed entity will be removed from the database at or before transaction commit or as a result of the flush operation.

Parameters
$entity : mixed

$flush : bool = false

Initiate a flush if true.

$soft : bool = false

For entities managed by Gedmo\SoftDeleteable first check if this entity is already tagged as "soft deleted" and if so do nothing.

Wrap the remove operation into a transaction or not. This parameter is ignored if $flush is false.
