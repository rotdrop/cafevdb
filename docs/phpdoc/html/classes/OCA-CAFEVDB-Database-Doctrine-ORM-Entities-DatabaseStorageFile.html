protected static array<int, EncryptedFile> $orphans = []

Array of potentially orphaned files which are no longer linked to dir-entries. The array is indexed by spl_object_id().

The actual work-horse for the orphans list is an entity listener which post-pones actual cleanup of orphans to a pre-commit handler of the decorated entity manager.
