Send to to a Cloud route.

public callInternalRoute(string $route[, string $method = self::POST ][, array<string|int, mixed> $routeParams = [] ][, array<string|int, mixed> $requestData = [] ][, string $postType = self::JSON ]) : array<string|int, mixed>
Parameters
$route : string

Route name or URL. If $route starts with a literal 'http' then it will be used as is. The $routeParams are ignored in this case and the $requestParams are appended to the URL for GET and posted to the URL for POST requests.

$method : string = self::POST
$routeParams : array<string|int, mixed> = []

Parameters built in to the URL (despite the fact that we use POST).

$requestData : array<string|int, mixed> = []

Stuff passed by the POST method.

$postType : string = self::JSON

How $requestData is encoded. Can be 'json' or 'urlencoded'. Default is 'json'.
