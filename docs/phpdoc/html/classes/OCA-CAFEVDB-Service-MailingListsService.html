configureAnnouncementsList()

MailingListsService.php : 505

Configure the given list as an announcements-only list.

public configureAnnouncementsList(string $listId[, string|array<string|int, mixed> $owners = [] ][, string|array<string|int, mixed> $moderators = [] ][, string|array<string|int, mixed> $posters = [] ]) : void

That is:

  • all postings are moderated
  • allow_list_posts set to false (no list-post header)
  • unsubscription is done without needing further confirmation from the user
  • subscription is unmoderated, but after user confirmation
  • $posters are added as list-members with the privilege to post to the list
  • the archive is public
Parameters
$listId : string

The list-id or FQDN. If a list-fqdn, then an additional query is needed to retrieve the list-id from the server.

$owners : string|array<string|int, mixed> = []

Array of owners or the single owner as string.

$moderators : string|array<string|int, mixed> = []

Array of moderators or the single moderator as string.

$posters : string|array<string|int, mixed> = []

Array of posters or the single posters as string. "posters" are the email addresses which are allowed to post to the list without moderation.
