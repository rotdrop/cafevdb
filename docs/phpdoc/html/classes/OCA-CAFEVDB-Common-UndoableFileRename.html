UndoableFileRename extends AbstractFileSystemUndoable
in package
  • cafevdb

UndoableFileRename.php : 45

Rename the given paths. The target path will be created if it does not exist. If the source path is empty the target folder will be created as empty folder.

If the target folder is empty, then the source folder will be deleted if it is empty. Non empty folders will bot be deleted.
